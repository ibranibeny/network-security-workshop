# Azure Network Security Workshop

This workshop demonstrates defense-in-depth in Azure with a hub-spoke topology, layered WAF, Azure Firewall Premium + IDPS, Bastion-only access, and centralized telemetry.

## Start Here

- [Workshop Walkthrough](docs/WORKSHOP-WALKTHROUGH.md)
- [Deployment Modules](README.md)
- [Teardown Guide](README.md#teardown)

## What You Will Learn

- How Front Door WAF and App Gateway WAF block attacks
- How Azure Firewall FQDN filtering and forced tunneling work
- How to validate segmentation and IDPS detections
- How to inspect evidence in Log Analytics and Azure Portal
